# Project Brief - Плодовка

## Название проекта
Саженцы плодовых культур "Сад и Огород"

## Цель проекта
Создать современный сайт-каталог для продажи саженцев плодово-ягодных культур с возможностью оформления заказа через Авито.

## Ключевые требования
- Интерактивный каталог саженцев с фильтрацией по категориям
- Поиск по сортам
- Корзина покупок
- Оформление заказа через Авито (копирование списка в буфер обмена)
- Адаптивный дизайн для мобильных устройств
- Тёмная/светлая тема
- PWA функциональность (офлайн-режим)

## Целевая аудитория
- Жители Севастополя и Крыма
- Жители других регионов РФ (доставка почтой/СДЭК)
- Любители садоводства, дачники
- Возраст: 25-65 лет

## Основные функции
1. Каталог саженцев (жимолость, смородина, крыжовник, малина)
2. Фильтрация по категориям
3. Поиск по названию/сорту
4. Детальные карточки товаров (фото, видео, характеристики)
5. Корзина с изменением количества
6. Оформление заказа через Авито
7. FAQ раздел
8. Контактная информация

## Успешные метрики
- Удобство использования на мобильных устройствах
- Скорость загрузки страницы
- Конверсия из посетителя в покупателя
- Положительные отзывы клиентов

## Ограничения
- Работа только через Авито (нет собственной оплаты)
- Данные загружаются из Google Таблицы
- Зависимость от внешних сервисов (Google Sheets, PapaParse CDN)
