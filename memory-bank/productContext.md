# Product Context - Плодовка

## Почему этот проект существует

Продажа саженцев плодово-ягодных культур через классические каталоги и объявления на Авито неудобна для покупателей:
- Нет единого места для просмотра всех сортов
- Сложно сравнить варианты
- Нет наглядной информации о наличии
- Процесс заказа разрознен

## Какую проблему решает

Сайт-каталог объединяет весь ассортимент в одном месте, позволяя:
- Быстро найти нужный сорт через поиск и фильтры
- Увидеть фото и характеристики каждого саженца
- Собрать заказ из нескольких позиций
- Отправить готовый список продавцу в один клик

## Как это работает

1. **Для покупателя:**
   - Открывает сайт → видит каталог саженцев
   - Фильтрует по категории или ищет сорт
   - Открывает карточку → смотрит фото/видео
   - Добавляет в корзину → оформляет заказ через Авито
   - Получает саженцы доставкой

2. **Для продавца:**
   - Получает готовый список заказа в Авито
   - Подтверждает наличие
   - Отправляет саженцы
   - Получает оплату

## Целевой пользовательский опыт

- **Первое впечатление:** понятно, что это магазин саженцев, сразу виден ассортимент
- **Навигация:** легко найти нужный сорт за 2-3 клика
- **Заказ:** минимальное количество действий для оформления
- **Доверие:** FAQ, гарантии, контакты, отзывы
- **Мобильность:** всё работает на телефоне без нареканий

## Ключевые сценарии использования

1. **Целевой:** "Хочу купить жимолость в Севастополе" → нашёл → добавил в корзину → заказал
2. **Исследование:** "Какие сорта смородины у вас есть?" → фильтр → просмотр → сравнение
3. **Повторный:** "Заказывал раньше, хочу ещё" → нашёл нужное → заказал
4. **Подарок:** "Хочу подарить саженец" → посмотрел FAQ → выбрал → заказал

## Эмоциональные триггеры

- **Уверенность:** гарантия приживаемости, проверенное качество
- **Удобство:** простой заказ, доставка до двери
- **Экспертность:** помощь в выборе сорта, рекомендации
- **Оптимизм:** красивый сад в будущем

## Конкурентные преимущества

- Удобный онлайн-каталог (конкуренты часто только фото в Авито)
- Гарантия приживаемости
- Прямая работа с производителями
- Быстрая обработка заказов
- Доставка по всей России
